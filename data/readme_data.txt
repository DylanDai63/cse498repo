Dataset: MNIST Handwritten Digits

Source: 
The dataset is automatically downloaded from the torchvision.datasets.MNIST module.

Download URL (fallback):
- http://yann.lecun.com/exdb/mnist/

Data Placement:
- The MNIST data will be automatically downloaded into the `data/` folder when running the code.
- No manual operation is needed. 
- If the download fails, please manually download the MNIST dataset from the above URL, and place the extracted files under the `data/` directory.

File Structure after download:
- data/MNIST/raw/train-images-idx3-ubyte
- data/MNIST/raw/train-labels-idx1-ubyte
- data/MNIST/raw/t10k-images-idx3-ubyte
- data/MNIST/raw/t10k-labels-idx1-ubyte
